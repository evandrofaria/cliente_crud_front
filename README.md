## Iniciando a aplicação
* **Verifique as versões**:
```sh
tsc -v
```
```sh
node -v
```
```sh
yarn -v
```
```sh
npm -v
```

* **Instale as dependências**:
```sh
yarn install
```
```sh
npm install
```

* **Inicie um servidor local**:
```sh
yarn start
```
